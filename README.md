# Bravimote - A cross-platform desktop and mobile remote control app for Sony Bravia TVs

Also check out the [original web-based version](https://github.com/wxllow/bravimote-web)

## Features

- Available on Windows, macOS, Linux, Android, and (TODO) iOS
- Automatically discover and pair TVs on the local network
- Full TV remote control functionality
- Full keyboard functionality (TODO)
- Customizable key bindings (TODO)
- Customizable UI
- Wake-on-LAN
- \[ANDROID\] Mateiral You Theming; \[WINDOWS] Accent Color Theming (TODO)
